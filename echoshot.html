Create Multiple Shots of Your Own Actor

Multi-Shot Text-to-Video Generation
Given multi-shot text prompts, EchoShot generates multiple video clips featuring the same character. It offers a highly flexible content controllability incuding duration, camera movement, human appearance, expressions, behaviors, background, ...











Multi-Shot Attribute Editing
EchoShot demonstrates an emergent capability of attribute editing. In a set of shots, it precisely edits certain attributes while keeping others unchanged, according to the given prompts.
